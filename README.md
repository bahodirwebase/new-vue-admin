## Configuration Options

- **Boxed Layout**: Containerized layout with max-width
- **Mini Sidebar**: Collapsible sidebar with mini mode
- **Shadow Theme**: Cards and components with shadows
- **Bordered Theme**: Cards and components with borders
- **Dark Mode**: Full dark theme support
